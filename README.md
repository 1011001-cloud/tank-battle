# 🎮 坦克大战 (Tank Battle)

经典 FC 坦克大战的 HTML5 网页复刻版，纯前端实现，无需任何依赖，打开即玩。

## 🕹️ 操作方式

| 按键 | 功能 |
|---|---|
| `↑ ↓ ← →` / `W A S D` | 移动坦克 |
| `空格` | 发射炮弹 |
| `P` | 暂停 / 继续 |
| `R` | 重新开始 |

## ✨ 游戏特性

- 🎯 **经典玩法** — 复刻经典 FC 坦克大战核心玩法
- 🧱 **可破坏地形** — 砖墙可被炮弹摧毁，钢墙不可摧毁
- 🌊 **水域障碍** — 坦克无法通过水域
- 🌳 **树林掩护** — 树林区域可以隐蔽坦克
- 🤖 **AI 敌人** — 敌方坦克具有自动移动和射击 AI
- 📈 **多关卡** — 击败所有敌人后进入更高难度关卡
- 💥 **粒子特效** — 爆炸、击中砖块产生粒子效果
- 🔊 **音效反馈** — 使用 Web Audio API 生成射击和爆炸音效
- 📊 **HUD 信息** — 显示生命、分数、剩余敌人数、关卡信息
- 🦅 **基地保护** — 守护地图下方的基地标志

## 🚀 快速开始

1. 克隆仓库：
   ```bash
   git clone https://github.com/1011001-cloud/tank-battle.git
   ```
2. 用浏览器打开 `index.html` 即可开始游戏

或者直接访问 GitHub Pages 在线游玩。

## 🛠️ 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6+)
- Web Audio API
- 零依赖，单文件实现
